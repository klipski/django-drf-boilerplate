# django-drf-boilerplate

## Description
This is a simple template repository for new projects.

## Installation
```
# create postgres volume
docker volume create --name=db_data

# install and run
docker-compose up -d
```

## Features
* django-rest-framework 3.12
* postgresql 13.1
* djoser 2.1
* redis 3.5
* celery 5.0
* beat 
* django-extensions
* swagger
